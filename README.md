## Clone the Repository 
`git clone https://github.com/luanluu168/data-scrubber.git`

## Configuration 
Create a .env file that looks like the .env.example file and fill in the environment variables. 

## Install Dependencies 
`npm install`

## To Run:
    `frontend: npm start`
    `backend: flask run`
